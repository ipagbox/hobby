## ✅ Content

- [ ] Все страницы имеют уникальный H1
- [ ] Все страницы имеют title (≤ 60 символов)
- [ ] Все страницы имеют meta description (≤ 155 символов)
- [ ] Контактная информация корректна (телефон, email, адрес)
- [ ] Нет placeholder текста ([TEXT], Lorem ipsum)
- [ ] Отзывы реальные, не выдуманные
- [ ] Цены актуальные (или убраны)

## ✅ SEO

- [ ] Sitemap.xml создан и доступен
- [ ] Robots.txt создан
- [ ] Canonical теги проставлены
- [ ] Alt тексты для всех изображений
- [ ] Schema markup: LocalBusiness добавлен
- [ ] Schema markup: Service добавлен для service pages
- [ ] Open Graph теги для социальных сетей
- [ ] Google Business Profile обновлён

## ✅ Frontend

- [ ] Сайт работает на мобильных (375px, 768px)
- [ ] Сайт работает на десктопе (1280px, 1920px)
- [ ] Все ссылки рабочие (нет 404)
- [ ] Изображения оптимизированы (WebP, < 200KB)
- [ ] LCP < 3 секунды
- [ ] CLS < 0.1
- [ ] Навигация работает корректно
- [ ] Формы отправляют данные

## ✅ Backend / Lead Capture

- [ ] Форма обратной связи отправляет письма
- [ ] Email уведомления настроены
- [ ] Данные форм сохраняются
- [ ] Защита от спама (reCAPTCHA или honeypot)
- [ ] SSL сертификат установлен

## ✅ Analytics

- [ ] Google Analytics / Яндекс.Метрика установлены
- [ ] Цели настроены (отправка формы, клик по телефону)
- [ ] Google Search Console подключён

## ✅ Legal

- [ ] Политика конфиденциальности создана
- [ ] Cookie consent (если нужен по GDPR/152-ФЗ)
- [ ] Согласие на обработку данных в форме

## ✅ Deployment

- [ ] Домен настроен
- [ ] DNS обновлён
- [ ] Хостинг настроен
