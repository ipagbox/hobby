# Discovery Report — Сантехника Мастер

**Stage:** 01-discovery
**Дата:** 2026-03-25
**Проект:** Сантехника Мастер, Москва

---

## 1. Анализ целевой аудитории

### Сегмент 1: Владельцы квартир (экстренные проблемы)
- **Описание:** Жители Москвы 30-55 лет, собственники или арендаторы квартир
- **Ситуация:** Прорвало трубу, потекла батарея, засорился слив — нужно решение сегодня
- **Боли:**
  - Страх затопить соседей
  - Не знают как найти надёжного мастера быстро
  - Опасаются, что мастер наживётся на срочности
- **Мотивация:** Скорость + честная цена + гарантия на работу
- **Поисковое поведение:** «сантехник срочно москва», «устранение течи срочно», «вызвать сантехника ЮЗАО»

### Сегмент 2: Владельцы квартир (плановый ремонт)
- **Описание:** Люди делающие ремонт или замену сантехники, 28-60 лет
- **Ситуация:** Замена труб, установка новой ванны/душевой кабины, монтаж тёплого пола
- **Боли:**
  - Сложно оценить объём и стоимость работ заранее
  - Страх некачественной работы и переделки
  - Нужно подстроить график мастера под свой
- **Мотивация:** Качество + прозрачность + опыт + портфолио аналогичных работ
- **Поисковое поведение:** «монтаж труб москва цена», «установка ванны москва», «замена сантехники квартира»

### Сегмент 3: Управляющие компании и ЖК
- **Описание:** Управдомы, ТСЖ, небольшие УК, ответственные за техобслуживание
- **Ситуация:** Нужен надёжный подрядчик для регулярных работ или разовых крупных задач
- **Боли:**
  - Нужна оперативность и надёжность
  - Важна документация (договор, акты)
  - Нужен партнёр на долгосрочной основе
- **Мотивация:** Надёжность + договор + возможность регулярного сотрудничества
- **Поисковое поведение:** «сантехнические работы юридическим лицам москва», «обслуживание сантехники ТСЖ»

**Главное УТП:** Бригада с 8-летним опытом, 500+ объектов, работают без выходных — надёжность + доступность.

## 3. Контекст локального рынка Москвы

- **Высокая конкуренция:** Рынок сантехнических услуг в Москве очень насыщен — множество частных мастеров, агрегаторы (YouDo, Профи.ру), крупные компании
- **Агрегаторы доминируют:** Большинство пользователей ищут через Профи.ру, YouDo, Авито — прямой поиск через сайт менее распространён для срочных задач
- **Доверие — ключевой фактор:** Клиенты выбирают по отзывам и портфолио, цена вторична
- **Локальное SEO критично:** Запросы с районом («сантехник ЮЗАО», «Юго-Запад») конвертируются лучше
- **Мобильный трафик преобладает:** 70%+ пользователей придут с телефона — номер должен быть кликабельным

## 4. Каналы привлечения (по приоритету для текущего бюджета)

1. **Органический поиск (SEO)** — долгосрочно, но критично для выживания без бюджета на рекламу
2. **Google/Яндекс Maps** — карточка компании (бесплатно, высокая конверсия)
3. **Сарафанное радио** — с сайтом как подтверждением экспертизы
4. **Контекстная реклама** — в будущем, при наличии бюджета

## 5. Гипотезы для проверки с клиентом

1. **H1:** Клиент получает большинство заказов через звонки, а не через форму — нужно проверить какой CTA конвертирует лучше (кнопка звонка vs форма обратного звонка)

2. **H2:** Сегмент «управляющие компании» приносит небольшой доход — стоит выяснить работает ли клиент с ТСЖ/УК и хочет ли развивать это направление

3. **H3:** Тёплые полы — нишевая и высокомаржинальная услуга — возможно стоит создать отдельную страницу для SEO

4. **H4:** Фото реальных работ у клиента есть — если нет, нужно планировать страницу «Портфолио» с placeholder и инструкцией по наполнению

5. **H5:** Клиент работает в основном в ЮЗАО, несмотря на «по всей Москве» — нужно уточнить реальную географию для настройки SEO и карт

## 6. Контентные приоритеты для сайта

На основе анализа ЦА и рынка, контент должен отвечать на:
- «Сколько стоит?» → прайс или диапазоны цен
- «Насколько быстро приедет?» → время выезда
- «Можно ли доверять?» → отзывы, фото, стаж, кейсы
- «Работают ли в мой район?» → ключевые слова с географией
- «Как вызвать?» → телефон и форма на каждом экране
