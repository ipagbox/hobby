## Task

Создай три файла: `launch-checklist.md`, `seo-audit.md`, `final-report.md`.

## Output: seo-audit.md

### SEO Audit по всем страницам

Для каждой страницы из sitemap:
```
| Страница | H1 | Title (длина) | Description (длина) | Keyword присутствует | Статус |
|---------|----|----|----|----|------|
| / | [H1 из page brief] | [title] (N chars) | [desc] (N chars) | ✅/❌ | OK/WARN |
```

### Технические SEO пункты
- Sitemap.xml: запланирован? ✅/❌
- Robots.txt: запланирован? ✅/❌
- Schema markup: какие типы запланированы?
- Canonical: стратегия определена?
- Internal linking: логика проверена?

### SEO Issues
Список найденных проблем с приоритетами: Critical / Warning / Info.

## Output: launch-checklist.md

Используй шаблон из `templates/launch-checklist.md`.
Заполни каждый пункт статусом из артефактов:
- ✅ = подтверждено в артефактах
- ⚠️ = частично / требует уточнения
- ❌ = не реализовано / не задокументировано
- 🔵 = требует ручной проверки (нельзя проверить без браузера)
